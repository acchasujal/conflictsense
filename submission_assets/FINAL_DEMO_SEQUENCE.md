# Final Demo Sequence

This document outlines the optimal ordering strategy for the live demonstration of ConflictSense. The goal is to build narrative tension and climax with a highly impactful, unexpected discovery that showcases true multi-step reasoning capabilities.

## 1. The Expected Findings

Begin the demo by showing ConflictSense identifying known, structural conflicts. These set the baseline for what the system can do and prove it understands cross-document relationships.

* **Compliance Conflict**: Demonstrate the system finding an overlap between **IT Security Policy §4.2** (requiring US-only processing) and the **DPDP Directive §3.1** (requiring Indian employee data to remain in India).
* **Data Governance Conflict**: Show how a broad data retention policy conflicts with a specific privacy request.

*Narration point:* "These are standard regulatory collisions. ConflictSense handles them instantly by retrieving the exact clauses via Azure AI Search."

## 2. The Climax: The Broken Anonymity Promise

The final reveal must demonstrate the system acting autonomously as a reasoning agent, finding a conflict that was never explicitly queried.

**The Conflict:** The **Whistleblower anonymity guarantee** explicitly promises employees that their identity will never be logged when submitting a report. However, the newly updated **Universal security logging** mandate requires that *all* internal network traffic, including the whistleblower portal, logs the user's IP and Active Directory ID.

*Narration script for this moment:*
> "Maya never asked ConflictSense to audit her anonymous reporting system. But because this is a Reasoning Agent, and not just a search tool, it understands the implications of the Universal Security Logging update."
> 
> "ConflictSense discovered the contradiction anyway. The anonymity promise is broken by the new security mandate. A standard search tool would never find this because the keywords don't match—only a Reasoning Agent can understand the logical collision."

## 3. The Resolution

Conclude the demo by highlighting the Human Approval Gate.

*Narration point:* "ConflictSense has identified a critical, high-risk liability. But it takes no action on its own. It presents the reasoning, the evidence, and the confidence score, routing it to a human for final approval."
